### 🌐 Web Development
* **[Portfolio](https://github.com/Rebanta-Gupta/Portfolio)**: Personal portfolio website built with React, TypeScript, and Vite.

### 📊 Data Science & Analysis Tools
* **[Statistical-Distribution-Fitting-Tool](https://github.com/Rebanta-Gupta/Statistical-Distribution-Fitting-Tool)**: Streamlit app for dataset visualization and fitting multiple statistical distributions using NumPy, Pandas, SciPy, and Matplotlib.

* **[taylor-vs-exact](https://github.com/Rebanta-Gupta/taylor-vs-exact)**: Interactive MATLAB tool for comparing Taylor series approximations vs exact function values with error analysis.

* **[Treasure-Game](https://github.com/Rebanta-Gupta/Treasure-Game)**: 2D vertical runner built with p5.js where you collect treasures and avoid obstacles.
* **[Bouncing-Ball-Simulation](https://github.com/Rebanta-Gupta/Bouncing-Ball-Simulation)**: Matter.js + p5.js physics simulation modeling high-restitution collisions.
